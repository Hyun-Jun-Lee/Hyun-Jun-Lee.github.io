# [DP] 개미전사

- 문제

<br>

- 입력 조건
  - 첫쨰줄에 식량창고의 개수 N이 주어진다.
  - 둘째 줄에 공백으로 구분되어 각 식량창고에 저장된 식량 개수 K가 주어진다.

- 입력 예시

```
4
1 3 1 5
```

- 출력 조건
  - 첫째줄에 개미 전사가 얻을 수 있는 식량의 최댓값을 출력

- 출력 예시

```
8
```

<br>

<summary>내 풀이</summary>
<div markdown="1">
<br>

```python
x=int(input())

a = list(map(int,input().split()))

# DP 테이블 초기화
dp=[0]*30001

# bottom-up 방식
dp[0] = a[0]
dp[1] = max(a[0],a[1])

for i in range(2,n):
  dp[i] = max(dp[i-1], dp[i-2]+a[i])

print(dp[n-1])
```
