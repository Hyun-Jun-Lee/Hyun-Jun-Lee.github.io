# 3. urls.py & view.py 

## 3.1 urls.py 

- 페이지 요청이 발생하면 가장 먼저 호출 되는 파일
- URL과 view 함수 간의 매핑을 정의(view 함수는 view.py 파일에 정의된 함수)

<br>

- urls.py 생성 후 app_name, path 등록
  - app_name은 html 작성시 `{% raw %}{% url 'accountapp:update' pk=user.pk %}{% endraw %}` 처럼 활용(스페이스바 주의!!!)
